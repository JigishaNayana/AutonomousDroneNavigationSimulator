# Autonomous Drone Navigation Simulator

A C++ simulation project demonstrating autonomous drone route planning in obstacle-rich environments.

## Features

- A* Pathfinding
- Obstacle Avoidance
- Battery Monitoring
- File-Based Map Loading
- Modular OOP Design
- Exception Handling
- STL Containers

## Folder Structure

DroneNavigator/

include/
src/
maps/
data/

## Build

g++ main.cpp src/*.cpp -I include -o drone

## Run

./drone

or

drone.exe

## Technologies

- C++
- STL
- OOP
- Priority Queue
- File Handling
- A* Search Algorithm
